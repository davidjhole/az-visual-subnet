❓ Help & Troubleshooting

📥 Importing from a live Azure environment

Use the export-azure.ps1 PowerShell script to export your existing Azure VNets and subnets into a JSON file compatible with this tool.

  1. Ensure you are logged in via az login (Azure CLI) or Connect-AzAccount (Az PowerShell)
  2. Run: .\export-azure.ps1 (optionally pass -SubscriptionId or -ResourceGroup to filter)
  3. The script outputs a JSON file — click Import JSON above to load it

📄 Copy for Word — pasting as plain text?

If the table pastes into Word as plain text instead of a formatted table, check your Word paste settings:

  1. In Word, go to File → Options → Advanced
  2. Scroll down to "Cut, copy, and paste"
  3. Set "Pasting from other programs" to "Keep Source Formatting" or "Merge Formatting"
  4. Click OK and try pasting again

Quick fix: In Word, right-click where you want to paste and choose the "Keep Source Formatting" paste option (clipboard icon with paintbrush), or use Paste Special → HTML Format.
